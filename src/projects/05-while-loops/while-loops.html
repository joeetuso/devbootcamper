While Loops

Practice creating while loops in JavaScript using the browser console

Problem Set

  1. Print all numbers between -10 and 19
  2. Print all even numbers between 10 and 40
  3. Print all odd numbers between 300 and 333
  4. Print all numbers divisible by 5 AND 3 between 5 and 50

Solutions

Problem #1

        var count = 1
        while (count < 19){
            console.log('count is: ' + count)
            count++
            }

Problem #2

        var count = 10
          while (count < 40){
          console.log('count is: ' + count)
          count += 2
        }

Problem #3

        var count = 301
          while (count < 334){
          console.log('count is: ' + count)
          count += 2
        }

Problem #4

        var count = 5
        while(count <= 50){
          if(count % 5 === 0
          && count % 3 === 0){
            console.log(count);
          }
          count+=1;
        }
