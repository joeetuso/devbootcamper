Functions

Practice creating functions in JavaScript using the browser console

Problem Set

  1. Write a function `isEven()` which takes a single numberic argument and rewturns true if the number is even, and false otherwise
  2. Write a function `factorial()` which takes a single numeric argument and returns the factorial of that number
  3. Write a function `kebabToSnake()` which takes a single kebab-cased string argument and returs the snake_cased version

Solutions

Problem #1

        function isEven(num){
          return num % 2 === 0;
        }

Problem #2

        function factorial(num) {
          let result = 1
          for (let i = 2; i <= num; i++) {
            result *= i
          }
          return result
        }

Problem #3

        function kebabToSnake(str) {
          // replace all '-' with "_"s
          const newStr = str.replace(/-/g, '_')
          return newStr
        }
